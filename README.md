
# Rating Film Web

Ini adalah project yang dibuat untuk memenuhi tugas proyek semester 2 quarter 2 untuk mata kuliah Web Application.

Aplikasi Web ini menampilkan daftar film yang ada pada database, deskripsi, aktor, rating, dan review dari pengguna lain.

Aplikasi ini menggunakan bahasa PHP untuk back-endnya, XAMPP untuk web server, dan MySql (MariaDB) untuk databasenya.
